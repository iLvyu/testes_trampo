# Verificador de Arquivos por Container

Script em Python que:
- Lê um Excel com números de container
- Verifica se existem arquivos correspondentes em uma pasta
- Marca "OK" no Excel
- Cria link clicável para abrir o arquivo
- Destaca pendências em amarelo

## Requisitos
- Python 3.10+
- pandas
- openpyxl

## Como usar
1. Ajuste os caminhos no script
2. Feche o Excel
3. Execute `python verifica_arquivo.py`
